Vim Command Reference

Basic Motion

Key Sequence Description
hMove cursor left one character
lMove cursor right one character
jMove cursor down one line
kMove cursor up one line
0Move cursor to the beginning of the line (column 0)
^Move cursor to the first non-blank character of the line
$Move cursor to the end of the line
g_Move cursor to the last non-blank character of the line
wMove cursor forward to the start of the next word
WMove cursor forward to the start of the next WORD (space-separated)
bMove cursor backward to the start of the previous word
BMove cursor backward to the start of the previous WORD (space-separated)
eMove cursor forward to the end of the current word
EMove cursor forward to the end of the current WORD (space-separated)
geMove cursor backward to the end of the previous word
gEMove cursor backward to the end of the previous WORD (space-separated)
ggGo to the first line of the file
GGo to the last line of the file
5GGo to line 5 (replace 5 with any number)
:5Go to line 5 (Ex command)
HMove cursor to the top of the screen (High)
MMove cursor to the middle of the screen (Middle)
LMove cursor to the bottom of the screen (Low)
f{char}Find next occurrence of {char} on the current line
F{char}Find previous occurrence of {char} on the current line
t{char}Move cursor *to* just before the next occurrence of {char}
T{char}Move cursor *to* just after the previous occurrence of {char}
;Repeat the last f, F, t, or T command
,Repeat the last f, F, t, or T command in the reverse direction
%Jump to matching parenthesis, bracket, or brace
(Jump to previous sentence
)Jump to next sentence
{Jump to previous paragraph
}Jump to next paragraph
[[Jump to previous function or block
]]Jump to next function or block
gjMove down one *display* line (for wrapped text)
gkMove up one *display* line (for wrapped text)
g0Move to the beginning of the *display* line
g^Move to the first non-blank char of the *display* line
g$Move to the end of the *display* line

Scrolling

Key Sequence Description
CTRL-FScroll forward one full screen
CTRL-BScroll backward one full screen
CTRL-DScroll down (forward) half a screen
CTRL-UScroll up (backward) half a screen
CTRL-EScroll down one line
CTRL-YScroll up one line
zzCenter the current line on the screen
ztMove current line to the top of the screen
zbMove current line to the bottom of the screen

Entering Insert Mode

Key Sequence Description
iEnter Insert mode before the cursor
IEnter Insert mode at the beginning of the line
aEnter Insert mode after the cursor
AEnter Insert mode at the end of the line
oOpen a new line below the current one and enter Insert mode
OOpen a new line above the current one and enter Insert mode
sSubstitute (delete) the character under the cursor and enter Insert mode
SSubstitute (delete) the entire current line and enter Insert mode (same as cc)

Insert Mode Commands

Key Sequence Description
CTRL-HDelete the previous character (like backspace)
CTRL-WDelete the previous word
CTRL-UDelete from the cursor to the start of the line
CTRL-TIndent the current line (add shiftwidth)
CTRL-DUn-indent the current line (remove shiftwidth)
CTRL-OExecute a single Normal mode command, then return to Insert mode
CTRL-PAutocomplete word (previous match)
CTRL-NAutocomplete word (next match)
CTRL-X CTRL-FAutocomplete file path
CTRL-X CTRL-LAutocomplete whole line
CTRL-V {char}Insert a special character literally (e.g., CTRL-V <Tab>)
<Esc>Exit Insert mode and return to Normal mode
CTRL-[Exit Insert mode (canonical alternative to <Esc>)

Search & Replace

Key Sequence Description
/patternSearch forward for 'pattern'
?patternSearch backward for 'pattern'
nRepeat last search in the same direction
NRepeat last search in the opposite direction
*Search forward for the whole word under the cursor
#Search backward for the whole word under the cursor
g*Search forward for the partial word under the cursor
g#Search backward for the partial word under the cursor
:s/old/new/Replace first 'old' with 'new' on the current line
:s/old/new/gReplace all 'old' with 'new' on the current line
:s/old/new/iReplace 'old' with 'new' on current line (case-insensitive)
:s/old/new/cReplace 'old' with 'new' on current line (with confirmation)
:%s/old/new/gReplace all 'old' with 'new' in the entire file
:%s/old/new/gcReplace all 'old' with 'new' in the file, with confirmation
:'a,'bs/old/new/gReplace all 'old' with 'new' between marks 'a' and 'b'
:.,+5s/old/new/gReplace all 'old' with 'new' from current line to 5 lines down
:g/pattern/s/old/new/gRun a substitution on all lines that match 'pattern'
:v/pattern/s/old/new/gRun a substitution on all lines that *do not* match 'pattern'
:nohClear search highlighting (no highlight)

Marks and Jumps

Key Sequence Description
m{a-z}Set mark {a-z} at cursor position (local to file)
m{A-Z}Set mark {A-Z} at cursor position (global, across files)
'{a-z}Jump to the *line* of mark {a-z} (first non-blank char)
`{a-z}`Jump to the *exact position* (line and column) of mark {a-z}
'{A-Z}Jump to the *line* of global mark {A-Z}
`{A-Z}`Jump to the *exact position* of global mark {A-Z}
''Jump to the *line* of the last jump
``Jump to the *exact position* of the last jump
'.Jump to the *line* of the last edit
`.Jump to the *exact position* of the last edit
giJump to the *exact position* of the last insert and enter insert mode
:marksList all current marks
CTRL-OGo to older position in the jump list (go 'back')
CTRL-IGo to newer position in the jump list (go 'forward')
:jumpsList all positions in the jump list
CTRL-]Jump to the definition of the tag under the cursor (uses tags file)
CTRL-TJump back from a tag definition

Registers

Key Sequence Description
"{register}yYank into a specific {register} (e.g., "ayy)
"{register}dDelete into a specific {register} (e.g., "add)
"{register}pPaste from a specific {register} (e.g., "ap)
"{register}PPaste before from a specific {register} (e.g., "aP)
"AyyAppend the current line to register 'a' (use uppercase letter)
"AddAppend-delete the current line to register 'a'
:regDisplay the contents of all registers
:reg {register}Display the contents of a specific {register}
""The unnamed register (default for y, d, p)
"0The yank register (holds the last yank, not overwritten by delete)
"1-9The numbered registers (hold last 9 deletes, 1 is newest)
"a-zThe named registers (user-controlled)
"A-ZThe named registers (appends to the lowercase version)
"*The system clipboard register (selection clipboard, X11)
"+The system clipboard register (primary clipboard)
".The last inserted text register
"%The current file name register
":The last Ex command register
"/The last search pattern register
"_The black hole register (deletes to here are gone forever, e.g. "_dd)
